# Security Policy

## Supported Versions

Orcho MCP is preparing its first public release line. Security fixes are
provided for the latest `0.1.x` release once `v0.1.0` is published.

Pre-release commits on `main` are not a supported distribution channel. Use
tagged releases and published packages for installable builds.

## Reporting a Vulnerability

Please report suspected security vulnerabilities through GitHub's private
vulnerability reporting flow for this repository.

Include as much detail as you can:

- affected version, commit, or package artifact;
- reproduction steps;
- expected and observed behavior;
- impact assessment;
- any known workaround.

Please do not disclose vulnerabilities publicly until a fix or mitigation is
available.

## Response Expectations

We aim to acknowledge new reports within seven days. For confirmed issues, we
will coordinate a fix, release timing, and public advisory text based on impact
and exploitability.
